# Hanse-Analyst

Dieses Projekt ist eine Demo-Anwendung, die meine Fähigkeiten in den Bereichen **Python**, **React** und **Künstliche Intelligenz (KI)** demonstriert.

## Projektziel
Hanse-Analyst ermöglicht es Benutzern, PDF-Dokumente hochzuladen, die anschließend von einer KI analysiert werden. Die Anwendung bietet eine Zusammenfassung des Inhalts und ermöglicht es dem Benutzer, spezifische Fragen zum Dokument zu stellen.

## Hauptfunktionen
- **PDF-Upload:** Einfaches Hochladen von Dokumenten über eine React-basierte Benutzeroberfläche.
- **KI-Analyse:** Automatische Extraktion und Verarbeitung von Informationen aus den hochgeladenen PDFs.
- **Frage-Antwort-System:** Interaktive Fragen zum Inhalt des Dokuments, die präzise beantwortet werden.
- **Dokumenten-Übersicht:** Schnelle Generierung eines Überblicks über die Kerninhalte.

## Datenschutz & Lokale KI
Ein zentraler Aspekt dieses Projekts ist der **Schutz der Privatsphäre**. 
Gerade für deutsche Unternehmen ist Datensicherheit von höchster Bedeutung. Daher arbeitet dieses Tool ausschließlich mit **lokal gehosteten KI-Clients**. 

- **Keine Cloud-Abhängigkeit:** Die Daten verlassen niemals die lokale Infrastruktur.
- **Maximale Datensicherheit:** Sensible Dokumente werden nicht an externe Server oder Drittanbieter übertragen.
- **Compliance:** Unterstützung bei der Einhaltung strenger Datenschutzrichtlinien (DSGVO).
